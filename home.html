What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, making it easier for developers to manipulate HTML documents. The DOM allows scripts, like JavaScript, to dynamically access, modify, and interact with the content, structure, and style of a document while it's loaded in a browser.

How do you select an element from the DOM?

To select elements from the DOM, JavaScript provides various methods that allow you to access and manipulate elements within an HTML document. Here are the most common ways to select DOM elements:

  • 1. getElementById()
  • 2. getElementsByClassName()
  • 3. getElementsByTagName()
  • 4. querySelector()
  • 5. querySelectorAll()
  • 6. getElementsByName()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element rather than adding event listeners to individual child elements. This listener can then respond to events triggered by its child elements due to event bubbling (or event propagation). It allows the parent element to "delegate" the handling of events that occur on its descendants, even if those descendants are added dynamically.
